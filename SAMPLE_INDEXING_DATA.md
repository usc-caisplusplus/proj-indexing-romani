## Data Entry Notes for Indexer

1. **Birth Dates:** Used `@ YYYY` format for estimated dates where exact date unknown
2. **Death Causes:** All Holocaust-related deaths explicitly attributed per PIQ testimony
3. **Geographic Terms:** All locations verified against city timeline definitions
4. **Name Handling:** 
   - Release name verified against signed Release Agreement
   - Aliases created for all documented spelling variations
   - Maiden names captured where applicable
5. **Relationships:** 
   - "Aunt, maternal" clarifies mother's side (not paternal)
   - "Uncle" (without paternal/maternal) used for in-laws
   - "Extended family member" used for distant cousin
6. **Place of Death Comments:** Used standardized abbreviations (POD, POBM, POBS, etc.)
7. **Proposed Terms:** All historical locations formatted correctly with country and location type
8. **Unverified Information:** Marked with asterisk (*) or noted in Notes field
9. **Unknown Information:** Left blank or marked as "Unknown" per guidelines
10. **PIQ vs. Testimony:** Entry reflects both PIQ documentation and audiovisual testimony when divergent
